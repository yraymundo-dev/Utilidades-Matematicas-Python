-Aplicacion desarrollada en Python con utilidades matematicas basicas, utilizando bucles y un menu interactivo para facilitar el uso.

FUNCIONALIDADES
-Determinar si un numero es par o impar.
-Calculo de factorial de un numero ingresado por el usuario.
-Tabla de multiplicar de un numero ingresado por el usuario.
-Menu interactivo para facilitar el uso.

TECNOLOGIAS UTILIZADAS
-Python 3.
-Programacion Estructurada.
-Uso de funciones, bucles y condicionales.

COMO EJECUTAR EL PROGRAMA
1- Clonar repositorio: git clone https://github.com/yraymundo-dev/Utilidades-Matematicas-Python.git
2- Acceder a directorio: cd Programacion
3- Ejecutar: python Refresh.py
